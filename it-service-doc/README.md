# 软件服务(外包)公司文档
> 如果没有产品，只做定制化开发的外包公司，很难平衡开发质量和利润，因为可复用的东西不多。有产品，可以针对的去找客户，代码质量可以不断打磨，成本也会越来越低。
> 业务成熟的公司，做定制化的需求一点。初创公司用现成的业务软件磨合业务。
